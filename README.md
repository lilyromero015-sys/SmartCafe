# SmartCafe

## Descripción
SmartCafe es una plataforma digital que ayuda a dentificar las enfermedades más comunes del café en el Cesar y recopilar información clara y comprensible sobre su prevención y manejo mediante el uso de la Inteligencia Artificial..

## Objetivo
Facilitar el acceso a información de plagas, ennfermedades y recomendaciones.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Python
- Inteligencia Artificial
- Git y GitHub
